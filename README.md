### [🔥 PythonMemorio](https://github.com/zhukovskaia/Memorio.git)

**Интерактивная платформа для изучения Python через геймификацию и интеллектуальное интервальное повторение (SRS).**

  * 🤖 [Адаптивный алгоритм интервального повторения, который автоматически планирует тренировки на основе ошибок и прогресса пользователя]
  * 📄 [Три игровых режима (Викторина, Спринт на время, Карточки) с системой уровней, XP, баллов и детальной статистикой успеваемости]
